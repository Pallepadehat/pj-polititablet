## Installation

Følg disse trin for at installere og bruge [Projektets navn]:

1. Klon dette repository:
   ```bash
   git clone https://github.com/Pallepadehat/pj-polititablet.git
   ```

2. Gå til projektmappen:
   ```bash
   cd pj-polititablet
   ```

3. Installér nødvendige afhængigheder:
   ```bash
   npm i
   ```

4. Start projektet:
   ```bash
   npm run dev
   ```
